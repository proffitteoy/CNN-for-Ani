# CNN for Ani 文档索引

本目录同时包含项目实验规范和冷启动前遗留的通用工程参考。执行任务时应先读项目实验规范，不要把通用模板机械套入项目。

## 核心实验文档

| 文件 | 角色 | 主要内容 |
|:---|:---|:---|
| [实现过程.md](./实现过程.md) | 主执行规范 | 原图采集、人工标注与 5 折划分、模型训练、权重导出和 Python/Kotlin 对齐 |
| [识别模型.md](./识别模型.md) | 理论基础 | 四位 multi-head classification、三层 CNN、损失、指标、增强与置信度 |

整个实验以 `实现过程.md` 的四阶段顺序推进；模型与数据决策必须满足 `识别模型.md` 的理论约束。若两者出现实际冲突，应先记录冲突并确认方案，不自行偏离。
